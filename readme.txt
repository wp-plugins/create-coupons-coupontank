=== CouponTank Coupons ===
Contributors: CouponTank
Tags: coupon, promotional, advertisements
Stable tag: trunk

Instantly displays coupons that you create using www.coupontank.com

== Installation ==
1. Download the plugin
2. Sign up for a www.coupontank.com account
3. Install your coupontank plugin
4. Open your plugin files and change the subdomain to your coupontank subdomain.

-- snippet
$coupontank = array(
  'subdomain' => 'sample'
);
